What is AI and ML?

Artificial Intelligence (AI) refers to the simulation of human intelligence in machines programmed to think and learn. Machine Learning (ML) is a subset of AI that allows systems to learn from data and improve their performance over time without explicit programming.

Project 1: Image Classification

In this project, you will create a model using Teachable Machine to classify images into different categories. You can choose to use a provided dataset or upload your own images.

Why This Project?

  • Learn the basics of image classification and machine learning.
  • Understand how to train and test AI models.
  • Gain hands-on experience with real-world applications.

Steps to Create the Model:

  1. Visit the Teachable Machine website.
  2. Choose "Image Project" and create two or more classes.
  3. Upload images to each class or capture images using your webcam.
  4. Train the model by clicking the "Train" button.
  5. Test the model with new images and export it as a TensorFlow.js model.

Download our sample dataset to get started quickly and begin your journey into the world of AI!

Project 2: Smart Image Gallery

In this project, you will build an interactive image gallery controlled by hand gestures using the model created in Project 1. This gallery allows you to showcase your images in a dynamic and engaging way.

Why This Project?

  • Explore gesture recognition and its applications.
  • Understand how to integrate AI models into web applications.
  • Create an engaging user experience with your gallery.

Steps to Create the Gallery:

  1. Train a gesture recognition model in Teachable Machine.
  2. Export the model and copy the model URL.
  3. Use the provided code to create a web application.
  4. Test the application by making gestures in front of the webcam.

Get started by downloading the code files or view the live site to see the gallery in action!
